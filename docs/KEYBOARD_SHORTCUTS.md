# Keyboard Shortcuts

English follows the Japanese.

[日本語](#日本語) | [English](#english)

## 日本語

`songcut` の編集画面では、セグメントの選択・境界調整・動画再生を
キーボードから操作できます。

![キーボードショートカット](image/keyboard-shortcuts.ja.png)

### セグメント編集

| キー | 操作 |
| --- | --- |
| `W` / `S` | 前／次のセグメントを選択 |
| `Q` / `E` | 最寄りの境界を左／右へ微調整 |
| `Z` / `X` / `C` | タイムラインを縮小／100%に戻す／拡大 |

### 動画再生

| キー | 操作 |
| --- | --- |
| `A` / `D` | 選択セグメントの開始／終了境界を再生 |
| `Space` | 再生／一時停止を切り替え |
| `Ctrl+A` / `Ctrl+D` | 前／次の境界へ移動 |

## English

In the `songcut` editor, you can select segments, adjust boundaries, and
control video playback from the keyboard.

![Keyboard shortcuts](image/keyboard-shortcuts.en.png)

### Segment Editing

| Key | Action |
| --- | --- |
| `W` / `S` | Select the previous / next segment |
| `Q` / `E` | Nudge the nearest boundary left / right |
| `Z` / `X` / `C` | Zoom the timeline out / reset to 100% / zoom in |

### Video Playback

| Key | Action |
| --- | --- |
| `A` / `D` | Play the start / end boundary of the selected segment |
| `Space` | Toggle play / pause |
| `Ctrl+A` / `Ctrl+D` | Jump to the previous / next boundary |
